вшитых числовых значений.

### Мета-прогрессия

| Значение | Цель |
|---|---|
| Уровней на улучшение | обычно отдельные одноуровневые ноды; повторяемые utility-бонусы могут иметь несколько уровней |
| Стоимость улучшений | зависит от роли апгрейда |
| Уникальный unlock cadence | примерно 1 unlock за first-clear уровень |
| Бонус скорости генерации | `-10% / -20% / -30%` |
| Бонус HP ядра | малый `+1`, большой `+10` |
| Значение регенерации HP | `1 / 2 / 3` HP в секунду |
| Версия прогресса | `3`; при несовпадении версии сохранение полностью очищается и создается заново |

Текущие базовые значения StingerMolecule, MembraneMolecule и SwarmMolecule представлены в `BattleMoleculeConfig`:
`1 / 1 / 1` атомов для создания связи Stinger/Membrane/Swarm,
`5` атомов для заряда StingerMolecule, `3` атома для заряда MembraneMolecule,
`5` секунд действия мембраны, потеря `1` секунды мембраны за каждую единицу
поглощённого урона, `3` базового урона и `2x` критического урона
StingerMolecule, `7` атомов для заряда SwarmMolecule, `1` базового урона
луча SwarmMolecule, `5` лучей роевого залпа, `30` градусов разброса,
`10` градусов дополнительного разброса за бонусный луч и `8` дальности
Swarm-залпа.
Таланты HP ядра, автогенерации ядра, скорости генерации снарядов кликами,
скорости connection-потока атомов, урона StingerMolecule, шанса критического
урона StingerMolecule, бонусной награды за критическое убийство, пробития,
прицела направления выстрела, расширения области
подбора ДНК, unlock MembraneMolecule, уменьшения заряда мембраны, длительности
мембраны, unlock SwarmMolecule, урона
SwarmMolecule, количества/ширины лучей SwarmMolecule, дальности SwarmMolecule
применяются в рантайме. Нода
дополнительной орбиты сейчас является конфигом дерева под следующий
gameplay-срез.
